# CodeViz

## Overview
CodeViz is an interactive platform for learning coding concepts through visualizations and debugging exercises.

## Key Components
- **Frontend**: Built with React.js for a dynamic user interface.
- **Backend**: FastAPI for high-performance web services.
- **Visualization Engine**: Renders 2D graphics using HTML5 Canvas or WebGL.
- **LLM Integration**: Provides advanced code analysis and debugging suggestions.
